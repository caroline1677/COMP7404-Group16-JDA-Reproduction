About JDA

Joint Distribution Adaptation (JDA) is a transfer learning method proposed by Long et al. (ICCV 2013). It aims to adapt source and target domains by jointly aligning both marginal and conditional distributions.

MMD: dist(Xs, Xt) = ||μs - μt||²

JDA minimizes the Maximum Mean Discrepancy (MMD) distance to reduce domain shift, while iteratively refining pseudo-labels for the target domain to adapt conditional distributions.

Marginal Distribution

Adapts P(Xs) ≈ P(Xt) using MMD

Conditional Distribution

Adapts P(Ys|Xs) ≈ P(Yt|Xt) iteratively

Feature Transformation

Learns a new feature representation A such that Z = ATX

Iterative Refinement

Uses 1-NN to generate pseudo-labels for target domain in each iteration
